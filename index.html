How to launch an app with (almost) no frameworks

There are some awesome frameworks out there – Bootstrap, React, Vue, Express, Sequelize... But sometimes, you do not need them to build an awesome app.

The upcoming Frameworkless set of video tutorials by @fiiv will cover all you need to go from 0 to a fully functional Node.js app, with almost no frameworks!

What will be covered?

  • An HTTP server using Node's own http module
  • Routing and templating
  • Using plain SQL instead of ORMs, and how to work it safely
  • Forms – not always needing Ajax!
  • Frontend assets – how to set up CSS and Javascript pre-processors
  • Does your frontend really need a framework to be "reactive"?
  • Finishing touches: Gzip, Pjax, and more
  • Writing tests with Node.js's `assert` library
